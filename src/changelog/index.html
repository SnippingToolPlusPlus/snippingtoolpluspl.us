The current stable release is Snipping Tool++ 5.3.5

Notice: Unfortunately we do not have a change log history for versions older than 5.0.1

5.3.5 Latest Stable Release

General Improvements

  • Editor now properly resizes
  • Editor preview color now correctly shows transparency level

Bug Fixes

  • Fixed a bug that would cause adding text to not appear on the final edit
  • Fixed a bug where a small rectangle would appear on the final image
  • Fixed a bug that caused the transparency slider to always reset

5.3.05th May 2013

General Improvements

  • Added straight line tool to the Editor
  • Changed the blur tools icon
  • You can now change the border color of rectangles
  • You can now change the stroke size for the editing tools

Bug Fixes

  • Snipping Tool++ was not clearing up unused memory correctly, which would result in huge memory consumption. This should no longer be an issue.

5.2.230th Mar 2013

Bug Fixes

  • No longer submits multiple uploads at a time when using the submit hotkey within the Editor
  • When using the text typing tool, text no longer disappears when you change the tool
  • Changing the color now correctly sets the transparency slider

5.2.120th Mar 2013

General Improvements

  • Snipping Tool++ now checks for new updates and notifies the user.
  • Various improvements to the new notification system

Bug Fixes

  • The 'ding' sound now correctly plays when an upload finishes

5.217th Mar 2013

General Improvements

  • A 'ding' sound is now played when an upload is finished
  • A new notification system has been implemented
  • Now using Java 1.6 instead of 1.7 previously

Bug Fixes

  • Fixed a few bugs on the Mac client, snipping should work correctly now

Known Bugs

  • Mac: When opening the popup menu, the menu sometimes opens on the left side of the screen, this is currently being looked into.

5.111th Mar 2013

General Improvements

  • Now has basic Mac OS functionality
  • Added Pastebin uploading feature. (Ability to log into a Pastebin accound has been removed)
  • Now comes packaged with a .exe wrapper for Windows users

Bug Fixes

  • Fix a minor bug that would sometimes happen when sending images to the Editor from the Capture Viewer
  • Various other minor fixes
